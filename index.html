3 製作指派與校稿

任務拆分

安排組員承接分會場活動頁＋外廣/站內AD/LinePush公版。視排程可拆開指派 (注意額外貼紙或特殊AD需求)。

版位與素材確認

與主管確認全站版位訊息，留意左上角小貼紙是否有提供，若無須提醒主管索取。與主管敲定給檔時間 (留意原始檔K數)。

過稿流程

  • 第一版 組長與主管過稿 -> OK後請組員分色 -> 於公版校稿窗與品類行銷確認分色 -> 提供分色psd 給承接ad製作的組員 -> 分會場活動頁＋外廣/站內AD/LinePush公版 完稿後再次給主管過目。
  • 主管看過沒問題後 -> 完成後協助雲端檔案檢查 -> 校稿窗通知品類行銷已完成-> 於組長窗提供公版連結給其他組長 -> 發佈給組內使用。
  • 若為先發設計組，需提供百貨組 第一版確認好的 PSD 檔。

4 換品與紀錄

  • 於相關 Chat 工作窗，公告該檔造節波段、入稿時間、負責設計，並 tag 品類行銷。
  • 填寫OGSM 表單： 檔期記錄填寫。
⚠️ 跨組溝通注意： 分會場公版視覺會與「百貨二組」輪流先發設計。製作期前務必先與百貨二組組長溝通好排程 (主視覺、AD公版交檔時間)。

EDM 作業流程

每月於 製播單 上查看需求並按走期時間安排人力排程。

數位：吳綺恩 家電：楊韻慈
及該檔所屬PM/企劃 (見PM通訊錄)

免費 EDM 流程

製播單表格「上方」區域為免費額度。

  1. 提前約 2~3 週於班表安排排程 (新品上線或其他特殊例外)。
  2. 若超出填寫時間仍無內容，提醒品類行銷補填。
  3. 開立 Chat 工作窗邀相關 PM/企劃入稿，Tag 負責設計。
  4. 回填製播單入稿時間，同步填入 OGSM 表單。
  5. 組員提出一校時，上雲端做視覺確認並按需求反饋調整修改。
  6. 案件結束後，確保前台網址與正確走期紀錄於 OGSM 表單。

付費 EDM 流程

商品部申請，製播單表格「下方」區域。

  1. 行銷找設計組長確認是否有時間安排製作。
  2. 請 PM/企劃 通知整銷單位 (陳苙萱) 進 BPM 申請付費。
  3. 整銷確認付費後通知設計主管，並由相關人員填入製播單。
  4. 組長依製播單安排人力與入稿時間，並填入 OGSM 表單。
重要：入稿時間前 3 天(不含假日)若未BPM申請並付費，直接抽單不製作！

「臨時需求」對應原則

1. 幫忙而非義務

臨時需求的製作物，屬於設計在工作允許的情況下主動協助的部分，非義務性工作。能否協助，將以設計是否有空檔時間為主要依據。

2. 排程優先順序

如遇常規製作物（包套、分會場、EDM）的排程與協助項目時間重疊，將以常規任務為優先，協助項目之時程將另行溝通調整。

3. 誠信原則

請勿在約定入稿日當天才告知要延遲。若需延後，請務必最晚提前 1 天說明。

4. 誠信紀錄評估

如多次未準時入稿，或協作過程中不斷追加工作量、反覆更改風格方向，將影響未來是否能再提供協助。

5. 校稿次數限制 (上限 3 次)

每件製作物提供最多 3 次校稿機會。請於每次回覆時「統整修改意見，避免零散提出」。

❌ 錯誤示範：早上改文案 → 下午調顏色 → 隔天補細節
⚠️ 處置方式：超過次數後，將提供原始檔請需求單位自行調整，設計不再進行後續。
