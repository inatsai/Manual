3 製作指派與校稿

任務拆分

安排組員承接分會場活動頁＋外廣/LinePush/站內AD/特效框公版。視排程可拆開指派 (注意額外貼紙或特殊AD需求)。

版位與素材確認

與主管確認全站版位訊息，留意分會場小貼紙是否有提供，若無須提醒主管索取。與主管敲定給檔時間 (留意原始檔K數)。

過稿流程

活動頁

  • 初版製作： 組長提供全站視覺psd參考 / 左上角貼紙公版 / 全站活動版位文案 給負責組員，進行風格延伸，分會場版頭設計，完成後提交初版視覺排版給主管確認。
  • 分色製作： 初版視覺排版確認後，按品類行銷提供的版型 / 色系需求，進行各分會場版型＋色系調整，完成後通知組長過稿，並於「全3C分會場_公版校稿窗」與品類行銷校稿確認，整理完成公版檔案。
  • psd檔案共享： 提交分色確認版本psd，給其他負責AD製作的組員，提供製作AD時使用。
  • 動態設計與規範說明： 活動頁執行動態設計與規範說明，製作並完稿後，一併上傳至公司雲端指定路徑空間（確保原始檔圖層整齊並留意K數大小）。
  • 若為先發設計組，需提供「百貨二組」初版確認好的PSD檔。

AD

  • 初版製作：收到活動頁分色確認psd檔後，進行風格色系延伸設計，完成後提交初版視覺給主管確認。
  • 分色製作： 初版視覺排版確認後，按各分會場psd檔內顏色，進行各分會場AD色系調整，完成後通知組長過稿，整理完成公版檔案，一併上傳至公司雲端指定路徑空間（確保原始檔圖層整齊並留意K數大小）。
  • AD項目： 分會場AD包含：外廣/Linepush/站內/特效框，如無特殊交代可沿用前一檔尺寸規格執行製作。
    ✱Line Push 口號需要做兩個版本，單口號＋雙口號（1040x1040 3品 / 1040x1040 4品 / 1040x1040 6品）。
    ✱Line Push 大標口號請統一使用思源黑體，以利商品部運用。
    ✱外廣需要做兩個版本，品堆版＋單品版。
  • 若為先發設計組，需提供「百貨二組」初版確認好的PSD檔。

分會場小貼紙：

  • 各活動頁左上角，皆需要押上，由全站提供公版貼紙，位置統一。
  • HeroBN 只有618、99購物節、1111、1212 ，大檔活動需要壓小貼紙。

該檔所屬PM/企劃 (見PM通訊錄)

免費 EDM 流程

製播單表格「上方」區域為免費額度（目前為每處每月兩封）。

  1. 提前約 2~3 週於班表安排排程 (新品上線或其他特殊例外)。
  2. 若超出填寫時間仍無內容，提醒品類行銷補填。
  3. 開立 Chat 工作窗邀相關 PM/企劃入稿，Tag 負責設計。
  4. 回填製播單入稿時間，同步填入 OGSM 表單。
  5. 組員提出一校時，上雲端做視覺確認並按需求反饋調整修改。
  6. 案件結束後，確保前台網址與正確走期紀錄於 OGSM 表單。

付費 EDM 流程

商品部申請，製播單表格「下方」區域為付費填寫區。

  1. 行銷找設計組長確認是否有時間安排製作。
  2. 請 PM/企劃 通知整銷單位 (陳苙萱) 進 BPM 申請付費。
  3. 整銷確認付費後通知設計主管，並由相關人員填入製播單。
  4. 組長依製播單安排人力與入稿時間，並填入 OGSM 表單。
重要：入稿時間前 3 天(不含假日)若未BPM申請並付費，直接抽單不製作！

包套類型差異

一般包套

視覺依 PM/企劃提出的風格設計（不跟公版）。不含高效媒體（外廣）尺寸。

額外(投廣)包套 = 高效媒體外廣

必須套用公版。前期需承接公版設計（與其他組輪流），窗口對整合行銷包套組於 包套 x Art廣告素材窗 承接入稿＋校稿。

*2026/01起新增 AD2 版位。

「臨時需求」對應原則

1. 幫忙而非義務

臨時需求的製作物，屬於設計在工作允許的情況下主動協助的部分，非義務性工作。能否協助，將以設計是否有空檔時間為主要依據。

2. 排程優先順序

如遇常規製作物（包套、分會場、EDM）的排程與協助項目時間重疊，將以常規任務為優先，協助項目之時程將另行溝通調整。

3. 誠信原則

請勿在約定入稿日當天才告知要延遲。若需延後，請務必最晚提前 1 天說明。

4. 誠信紀錄評估

如多次未準時入稿，或協作過程中不斷追加工作量、反覆更改風格方向，將影響未來是否能再提供協助。

5. 校稿次數限制 (上限 3 次)

每件製作物提供最多 3 次校稿機會。請於每次回覆時「統整修改意見，避免零散提出」。

❌ 錯誤示範：早上改文案 → 下午調顏色 → 隔天補細節
⚠️ 處置方式：超過次數後，將提供原始檔請需求單位自行調整，設計不再進行後續。
